# 資源列表

> 維護說明
>
> - 兩層結構:外層用 `## 區塊` (共用 / team:a / team:b),內層用 `### 分類` (學習 / 工具 / 參考...)
> - 區塊標題用 **內部 ID** (`team:a`、`team:b`、`共用`),不是顯示名 (MapMo / Diula)。顯示名由儀表板從 `teams.json` 查表。
> - 每筆資源用一行清單格式:`- [標題](URL) — 說明`
> - 說明可以省略,但建議寫一句話讓組員知道為何要看。
> - 加新資源 → 直接編輯這個檔案 → push → 各組頁面自動更新。
